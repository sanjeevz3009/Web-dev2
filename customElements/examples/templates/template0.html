Template Example 0

In this example, a single HTML template t1 is defined, but it's not used immediately.

The script gets a handle on the template and clones it.

The content of the original template and its clone (which are both DocumentFragments) can be inspected in the console and seen to be the same in structure, indicating that the cloning process has worked.

The cloned node can be manually edited:

cloned.firstElementChild.textContent='This text was added.';

In the console, inspect cloned and t1 to see how they now differ, then add them to the document:

document.body.append(cloned);
